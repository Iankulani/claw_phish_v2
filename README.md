# claw_phish_v2

Claw Phish V2  a cyber security sofware used for penetraion testing, cyber drills, cyber research, black hat hackers, white hat, Red team and for education purpose the tool enable the user to fire commands via Discord, Slack, Whtsapp, iMessage, Google Chat, Signal and Web Application.


# How to clone the repo
```bash

git clone https://github.com/Iankulani/claw_phish_v2.git
cd claw_phish_v2
```

# How to run

```bash
python claw_phish_v2.py
```
